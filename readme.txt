It's a WooCommerce extension allowing clients make payment with zipwallet.

== Description ==
WooCommerce payment gateway plugin for zipwallet which accept payments from client through M-Pesa, Airtel, Equitel, VISA, Mastercard and Pesalink. Fully supported with WordPress and WooCommerce latest versions. zipwallet latest API 3.0 compatible.

= Features =
* Enable and disable various payment channels.
* Enable/Disable Test Mode

= How to Setup Account with zipwallet = 
[Step by step zipwallet Account Setup Guide](https://zipwalletpay.com/developers/)

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, you can set options from `WooCommerce -> Settings -> zipwallet` menu
